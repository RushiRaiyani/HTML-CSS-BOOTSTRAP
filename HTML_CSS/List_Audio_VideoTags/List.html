Concept of WWW

  • It stands for world wide web
  • It is basically a system of internet servers that support specially formated documents.
  • It is a collection of text pages,videos,digital photographs and animation we can access over the internet.
  • User access the World Wide Web facilties via client called a web browser.

HTTP Protocol

  1. HTTP stands for Hyper Text Transfer Protocol.
  2. HTTP is based on client server architecture model and a stateless request/respose protocol
  3. 404 File Not Found is common HTTP status code.
